## Дополнительные материалы

### Рекомендуемые терминальные эмуляторы
- macOS: iTerm2, Terminal.app
- Linux: GNOME Terminal, Konsole, Tilix
- Windows: Windows Terminal, PowerShell, WSL

### Полезные ресурсы
- [The Linux Command Line](http://linuxcommand.org/tlcl.php) - книга по основам командной строки
- [ExplainShell](https://explainshell.com/) - объяснение команд Linux
- [Oh My Zsh](https://ohmyz.sh/) - фреймворк для управления конфигурацией Zsh

### Сочетания клавиш для быстрой работы
- Ctrl+A/Ctrl+E - начало/конец строки
- Ctrl+R - поиск по истории
- Ctrl+L - очистка экрана
- Ctrl+C - прерывание команды
- Ctrl+D - выход из сессии
