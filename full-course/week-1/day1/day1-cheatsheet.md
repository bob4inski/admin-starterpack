# Шпаргалка по командам терминала для Дня 1

## Навигация по файловой системе

| Команда | Описание | Пример |
|---------|----------|--------|
| `pwd` | Показать текущую директорию | `pwd` |
| `cd` | Сменить директорию | `cd /home/user` |
| `cd ..` | Перейти в родительскую директорию | `cd ..` |
| `cd ~` | Перейти в домашнюю директорию | `cd ~` |
| `cd -` | Перейти в предыдущую директорию | `cd -` |
| `ls` | Показать содержимое директории | `ls` |
| `ls -l` | Показать содержимое с подробной информацией | `ls -l` |
| `ls -a` | Показать все файлы, включая скрытые | `ls -a` |
| `ls -la` | Комбинация -l и -a | `ls -la` |
| `tree` | Показать структуру директорий в виде дерева | `tree` |

## Работа с файлами и директориями

| Команда | Описание | Пример |
|---------|----------|--------|
| `mkdir` | Создать директорию | `mkdir new_dir` |
| `mkdir -p` | Создать директорию со всеми родительскими | `mkdir -p path/to/dir` |
| `rmdir` | Удалить пустую директорию | `rmdir empty_dir` |
| `rm` | Удалить файл | `rm file.txt` |
| `rm -r` | Удалить директорию и её содержимое | `rm -r dir` |
| `rm -f` | Удалить без подтверждения | `rm -f file.txt` |
| `cp` | Копировать файл/директорию | `cp source.txt dest.txt` |
| `cp -r` | Копировать директорию рекурсивно | `cp -r source_dir dest_dir` |
| `mv` | Переместить/переименовать файл/директорию | `mv old.txt new.txt` |
| `touch` | Создать пустой файл или обновить время доступа | `touch new_file.txt` |

## Просмотр содержимого файлов

| Команда | Описание | Пример |
|---------|----------|--------|
| `cat` | Показать содержимое файла | `cat file.txt` |
| `less` | Просмотр файла с прокруткой | `less file.txt` |
| `more` | Просмотр файла с прокруткой (базовый) | `more file.txt` |
| `head` | Показать первые строки файла | `head -n 10 file.txt` |
| `tail` | Показать последние строки файла | `tail -n 10 file.txt` |
| `tail -f` | Показывать последние строки в реальном времени | `tail -f log.txt` |
| `wc` | Подсчитать строки, слова и символы | `wc file.txt` |
| `wc -l` | Подсчитать только строки | `wc -l file.txt` |

## Поиск файлов и текста

| Команда | Описание | Пример |
|---------|----------|--------|
| `find` | Найти файлы по различным критериям | `find . -name "*.txt"` |
| `grep` | Искать текст в файлах | `grep "pattern" file.txt` |
| `grep -r` | Рекурсивный поиск в директориях | `grep -r "pattern" .` |
| `grep -i` | Регистронезависимый поиск | `grep -i "pattern" file.txt` |
| `grep -n` | Показать номера строк | `grep -n "pattern" file.txt` |
| `locate` | Найти файлы по имени в базе | `locate filename` |
| `which` | Найти путь к исполняемому файлу | `which python` |
| `whereis` | Найти бинарные, исходные и man-страницы | `whereis python` |

## Полезные символы и шаблоны

| Символ | Описание | Пример |
|--------|----------|--------|
| `*` | Любое количество символов | `ls *.txt` |
| `?` | Один любой символ | `ls file?.txt` |
| `[]` | Любой символ из указанных | `ls file[123].txt` |
| `>` | Перенаправить вывод в файл (перезапись) | `ls > files.txt` |
| `>>` | Перенаправить вывод в файл (добавление) | `ls >> files.txt` |
| `<` | Ввести данные из файла | `command < input.txt` |
| `|` | Передать вывод другой команде | `ls -l | grep ".txt"` |
| `&` | Выполнить команду в фоне | `command &` |
| `;` | Разделитель команд | `cmd1; cmd2` |
| `&&` | Выполнить вторую команду, если первая успешна | `cmd1 && cmd2` |
| `||` | Выполнить вторую команду, если первая неудачна | `cmd1 || cmd2` |

| `Ctrl+b p` | Переключиться на предыдущее окно |

| `Ctrl+b %` | Разделить окно вертикально |
| `Ctrl+b "` | Разделить окно горизонтально |
| `Ctrl+b o` | Переключиться на следующую панель |
| `Ctrl+b x` | Закрыть текущую панель |
