# Практические задания для Дня 1: Настройка терминала и основы командной строки

## Задание 1: Настройка терминала

### Цель
Настроить удобную рабочую среду в терминале

### Задачи
1. Установите терминальный эмулятор (если необходимо):
   - macOS: iTerm2
   - Linux: Tilix или Terminator
   - Windows: Windows Terminal

2. Настройте цветовую схему:
   - Выберите тёмную тему (например, Dracula, Monokai, или Material)
   - Настройте контрастность для комфортной работы

3. Настройте шрифт:
   - Установите моноширинный шрифт (Fira Code, JetBrains Mono, или Source Code Pro)
   - Установите размер шрифта 12-14pt

4. Изучите и настройте 10 основных сочетаний клавиш:
   - Ctrl+A/Ctrl+E - переход в начало/конец строки
   - Ctrl+R - поиск по истории команд
   - Ctrl+L - очистка экрана
   - Ctrl+C - прерывание текущей команды
   - Ctrl+D - выход из сессии
   - Ctrl+Z - приостановка текущей команды
   - Ctrl+U - удаление строки до курсора
   - Ctrl+K - удаление строки после курсора
   - Ctrl+W - удаление слова до курсора
   - Tab - автодополнение

### Проверка
- [ ] Терминал установлен и настроен
- [ ] Цветовая схема подобрана
- [ ] Шрифт установлен и настроен
- [ ] Сочетания клавиш изучены и работают

## Задание 2: Настройка оболочки

### Цель
Настроить командную оболочку для удобной работы

### Задачи
1. Выберите оболочку (Bash или Zsh) и настройте её:
   - Установите Zsh (если выбрали его)
   - Сконфигурируйте .bashrc или .zshrc

2. Настройте алиасы:
   - ll для ls -alF
   - la для ls -A
   - .. для cd ..
   - grep с цветным выводом

3. Настройте промпт:
   - Отображение текущего пользователя
   - Отображение текущей директории
   - Цветовое выделение

4. Создайте полезные функции:
   - mkcd - создание директории и переход в неё
   - extract - извлечение архивов различных форматов

### Проверка
- [ ] Оболочка настроена
- [ ] Алиасы работают
- [ ] Промпт настроен
- [ ] Функции работают корректно

## Задание 3: Работа с файловой системой

### Цель
Освоить базовые команды для работы с файловой системой

### Задачи
1. Создайте структуру директорий:
   ```
   admin-course/
   ├── week1/
   │   ├── day1/
   │   │   ├── lecture/
   │   │   ├── practice/
   │   │   └── homework/
   │   └── day2/
   │       ├── lecture/
   │       ├── practice/
   │       └── homework/
   ├── resources/
   ├── scripts/
   └── configs/
   ```

2. Перемещайтесь между директориями:
   - Используя абсолютные пути
   - Используя относительные пути
   - Используя автодополнение

3. Создайте файлы:
   - В каждой директории создайте файл README.md с описанием назначения директории
   - В директории scripts создайте файл backup.sh
   - В директории configs создайте файл .vimrc

4. Работайте с файлами:
   - Скопируйте файл README.md из day1 в day2
   - Переместите файл backup.sh в директорию day1/practice
   - Переименуйте файл .vimrc в vimrc-config

### Проверка
- [ ] Структура директорий создана
- [ ] Навигация работает
- [ ] Файлы созданы
- [ ] Операции с файлами выполнены

## Задание 4: Поиск и фильтрация

### Цель
Научиться искать файлы и фильтровать вывод команд

### Задачи
1. Поиск файлов:
   - Найдите все файлы с расширением .md
   - Найдите все директории
   - Найдите файлы, созданные за последние 10 минут

2. Поиск текста в файлах:
   - Найдите все строки, содержащие "admin" в файлах .md
   - Найдите все строки, начинающиеся с "#"
   - Подсчитайте количество строк во всех файлах .md

3. Фильтрация вывода:
   - Выведите список файлов, отсортированный по размеру
   - Выведите список файлов, отсортированный по времени изменения
   - Выведите только имена файлов без путей

4. Комбинирование команд:
   - Найдите все .md файлы и подсчитайте общее количество строк
   - Найдите все файлы, содержащие "config", и выведите их размеры
   - Создайте список всех директорий и сохраните его в файл

### Проверка
- [ ] Команды поиска работают
- [ ] Поиск текста работает
- [ ] Фильтрация работает
- [ ] Команды успешно комбинируются
